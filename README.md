# TrayClash

**TrayClash** — это легкий и минималистичный клиент для [mihomo](https://github.com/MetaCubeX/mihomo) (Clash Meta), работающий исключительно из системного трея Windows. Программа объединяет в себе мощь ядра Clash и простоту управления без громоздких интерфейсов.

## Основные возможности

- **Всё в одном:** Ядро `mihomo` и драйвер `wintun.dll` встроены прямо в исполняемый файл. Никаких лишних файлов в папке с программой.
- **Управление из трея:** Запуск и остановка сервиса в один клик.
- **Выбор прокси:** Быстрое переключение между серверами прямо из контекстного меню (поддерживаются группы типа `selector`).
- **Менеджер подписок:** Поддержка нескольких подписок с возможностью добавления, удаления и переключения между ними.
- **Автозагрузка:** Возможность автоматического запуска программы при старте Windows.
- **Zashboard:** Быстрый доступ к современной веб-панели управления для детальной настройки и мониторинга.
- **Портативность:** Все настройки и скачанные конфигурации хранятся в `%APPDATA%\TrayClash`.

## Системные требования

- **ОС:** Windows 7 или новее (x86 / x64).
- **Важное примечание для Windows 7:**
  Для работы режима TUN (и драйвера `wintun.dll`) **обязательно** наличие обновления **[KB4474419](https://www.catalog.update.microsoft.com/Search.aspx?q=KB4474419)**. Оно добавляет поддержку подписи кода SHA-2, без которой драйвер не сможет загрузиться.

## Использование

1. Скачайте последнюю версию со страницы релизов.
2. Запустите `TrayClash.exe`.
3. При первом запуске (или через меню «Подписки») добавьте URL вашей подписки.
4. Нажмите **«Включить»** в меню трея.
5. Выберите нужный сервер в подменю **«Прокси»**.

## Сборка из исходников

Если вы хотите собрать проект самостоятельно, вам потребуется установленный [Go](https://go.dev/) и утилита [go-winres](https://github.com/tc-hib/go-winres).

1. Установите `go-winres`:
   ```bash
   go install github.com/tc-hib/go-winres@latest
   ```
2. Запустите скрипт сборки:
   ```bash
   build.bat
   ```
3. Готовые файлы появятся в папке `dist/x64` и `dist/x86`.
